# 🚀 Dion's Portable Dev-Station

My personal daily-driver lab environment on Debian. Managed with Docker, secured with Nginx Proxy Manager.

## 🛠 Core Services
- **Nginx Proxy Manager**: Reverse proxy for local `.local` domains.
- **AdGuard Home**: DNS management and network-wide ad blocking.

## 🤖 Apps & Bots
- **Photo to WebP Converter**: Telegram bot for image optimization.

## 📂 Project Structure
- `core/`: Critical infrastructure services.
- `apps/`: Custom development projects and utilities.

## 🚀 How to Run
`docker compose up -d`
